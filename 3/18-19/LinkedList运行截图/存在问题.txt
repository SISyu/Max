2.结构体定义有无typedef的区别：
      当只是定义结构体类型：

	struct 结构体名

	{
  	     结构体成员名

	}结构体变量名；

      引用结构体成员方式应是：

	结构体变量名. 结构体成员名  //结构体变量直接引用结构体成员

      当使用typedef定义时：

	typedef struct 结构体名

	{
    	    结构体成员名

	}结构体类型名；

      引用结构体成员方式应是：

	结构体类型名 xxx；// 定义一个结构体变量

	xxx.结构体成员名  // 引用结构体成员
